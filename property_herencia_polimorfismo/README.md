# @property + Herencia + Polimorfismo
---
Seria bueno tener claro el concepto de **encapsulamiento**
y significa proteger el estado interno del objeto y controlar como se accede o modifica

En POO avanzado o profesional:
- Los atributos no deberian modificarce.
- Se debe validar antes de cambiar un valor.
- El objeto es respnsable de mantener su propia coherencia.
